## Employee Management System – Spring MVC CRUD Application

A simple CRUD (Create, Read, Update, Delete) web application built using Spring MVC, Spring Boot, Thymeleaf, JPA, and MySQL. This project allows you to manage employees by adding, listing, updating, and deleting employee records.

## Features

Add a new employee

View all employees

Update existing employee details

Delete an employee

Beautiful UI using CSS

MVC architecture with Service + Repository layers

Fully responsive views with Thymeleaf Templates

## Technologies Used

Java 17+

Spring Boot (Spring MVC + Spring Web)

Spring Data JPA (Hibernate)

MySQL Database

Thymeleaf

Maven

HTML5, CSS3

## How to Run the Application

1. Clone the Repository

2. Configure MySQL

3. Run the Application

4. Open in Browser

## CRUD Endpoints

| Operation       | URL                      | Method |
| --------------- | ------------------------ | ------ |

| List Employees  | `/employees`             | GET    |

| Show Add Form   | `/employees/showForm`    | GET    |

| Add Employee    | `/employees/save`        | POST   |

| Update Employee | `/employees/update?id=1` | GET    |

| Delete Employee | `/employees/delete?id=1` | GET    |
